# AIMWAER注入动画
一个简单的脚本 能够实现AW在注入后的动画效果。原版AW注入无声无息的 于是有了这个想法<br>
一开始想使用logo旋转实现的，但是AIMWARE for CS2中没有了引入图片的方法，尝试使用外链/编码/SVG多种方法都失败了 最后选择了ASCII文字画<br>
你可以将此lua设置为自动加载（autorun.lua），以便在注入时启用动画。

下面的图片都为静态，实际有动效。具体效果请亲自体验~
